# AI Inference Engine
This is an inference engine for the llama3.2 1B F16 models. It is completely written in C#.

## How to Install?
Downlaod an llama3.2 1B F16 .gguf file. The following hugging-face source is recommended: https://huggingface.co/bartowski/Llama-3.2-1B-Instruct-GGUF/blob/main/Llama-3.2-1B-Instruct-f16.gguf
The project comes pre-installed. However, it requires the correct .Net versions to run (Visual Studio is recommended).

## How to Run?
Run the LLMInference project.
